## Tech Stack

**Client:** React, Chakra UI

**Server:** -
## Run Locally

Clone the project

```bash
  git clone https://github.com/imrod22/coderhouse-react-plant-b-project.git
```

Go to the project directory

```bash
  cd plantb
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
